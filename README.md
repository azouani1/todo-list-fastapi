# ToDo List - Projet de rattrapage INFOB238

Application web FastAPI permettant à un utilisateur :
- d’ajouter une tâche 
- de supprimer une tâche 
- de voir toutes les tâches 

## Lancer le projet
1. Cloner le repo
2. Installer les dépendances :
    pip install -r requirements.txt
3. Lancer :
    uvicorn app.main:app --reload
